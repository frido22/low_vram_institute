# lab_public

This is the public artifact stream for the autonomous lab.

The public repo is intentionally noisy. Each run gets a small package with the experiment summary, metrics, patch, provenance, and updates to the public agenda and leaderboard.

See:

- `public/current_status.md`
- `public/agenda.md`
- `public/latest_thoughts.md`
- `public/leaderboard.md`
- `runs/ledger.jsonl`
